# LenZzyKid Site

A professional photo storefront for selling aerial images, photo drops, prints, and custom visual work.

Open `index.html` in a browser, or run a local static server from this folder.

## Customize

- Add new photos to `assets/` and update the image paths in `index.html`.
- Replace the email links with your real store, payment, or contact links.
- Update the email address in the contact section.
